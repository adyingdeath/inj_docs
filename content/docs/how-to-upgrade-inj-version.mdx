---
title: How to Upgrade INJ Version
description: A guide on how to upgrade INJ to the latest version.
---

import { Steps, Step } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
First, download the latest version of INJ from the [release page](https://github.com/adyingdeath/INJ/releases/latest).
</Step>

<Step>
Then, unzip the file and replace all the files in the old version with the new version.
</Step>

</Steps>

In most cases, you don't need to run `install.js` again. However, if you come across any errors, you can run `install.js`, or repeat the [installation](/docs/quick-start/#installation) process again.

If you have any questions, please join our discord server for help.
